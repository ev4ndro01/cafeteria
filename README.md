# Coffee Shop - Landing Page

Uma landing page moderna e responsiva desenvolvida para uma cafeteria, utilizando **Bootstrap 5**. O projeto apresenta uma interface elegante com carrossel de imagens, seções informativas sobre a equipe e um cardápio interativo organizado por abas.

## 🚀 Funcionalidades

* **Design Responsivo:** Adaptável a diferentes tamanhos de tela (Mobile, Tablet e Desktop).[cite: 1]
* **Barra de Navegação Fixa:** Navegação facilitada com menu hambúrguer para dispositivos móveis.[cite: 1]
* **Carrossel Dinâmico:** Destaques da cafeteria com transições automáticas.[cite: 1]
* **Seção 'Sobre Nós':** Layout em cards para apresentar a paixão pelo café e a equipe de baristas.[cite: 1]
* **Cardápio Interativo:** Navegação por abas (Tabs) para separar categorias como Expressos, Capuccinos, Gelados e Especiais.[cite: 1]
* **Localização via Modal:** Botão que abre um modal com informações de endereço e um mapa integrado.[cite: 1]
* **Rodapé Informativo:** Contatos e identidade visual da marca.[cite: 1]

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação do conteúdo.[cite: 1]
* **CSS3:** Estilizações personalizadas (através do arquivo `src/css/style.css`).[cite: 1]
* **Bootstrap 5.3.x:** Framework para layout, componentes e responsividade.[cite: 1]
* **Unsplash API:** Imagens de alta qualidade utilizadas no projeto.[cite: 1]
